# Модуль для работы с окружностями

## Общее описание
Модуль предоставляет функции для вычисления основных геометрических характеристик окружности: площади и длины окружности (периметра).

## Описание функций

### area(r)
Назначение: Вычисляет площадь окружности по заданному радиусу.

Параметры:
- r (float/int): радиус окружности

Возвращаемое значение:
- float: площадь окружности, вычисленная по формуле π × r²

Пример использования:
`python
import math

# Вычисление площади окружности с радиусом 5
```
result = area(5)
print(result)  # Вывод: 78.53981633974483
```

# Модуль для работы с квадратами

## Общее описание решения
Модуль предоставляет функции для вычисления основных геометрических характеристик квадрата. 
Реализованы математические операции для расчета площади и периметра квадрата по заданной длине стороны.

## Математические формулы
- Площадь квадрата: A = a²
- Периметр квадрата: P = 4 × a

где:
- a - длина стороны квадрата

## Использование
```python
from square import area, perimeter

# Вычисление площади квадрата со стороной 5
print(area(5))  # Вывод: 25

# Вычисление периметра квадрата со стороной 5
print(perimeter(5))  # Вывод: 20
```
